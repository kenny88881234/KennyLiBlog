[問題] C# 如何使用全域變數

在 C# 想宣告全域變數，在最外層直接宣告是不行的，C# 中並沒有宣告全域變數的方式，一般都是透過屬性 ( property ) 或靜態變數 ( static ) 來達成

靜態變數 ( static )

屬性 ( property )

  • 屬性的好處

    • 權限控制 : 在 set 前加上修飾詞 private 或 internal，代表此變數只有自己或內部能修改
    • 行為控制 : 可以在屬性判斷傳入的值是否符合規則、是否要觸發事件等等
